A simplistic weather station written in Typescript running on a Raspberry Pi using Node.js.

When activated by a user using a hardware button, the application queries latest weather information from multiple sources, extracts three key data-points (is it raining, windy and/or cold), and generates a sound file using the Microsoft Speech Synthesis API based on resulting answers to be played to the querying user.
